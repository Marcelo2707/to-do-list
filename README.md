# ✅ To Do Avançado

Aplicação web de lista de tarefas com funcionalidades avançadas: criação, edição, exclusão, pesquisa e filtro de tarefas.

## 📋 Sobre o projeto

Um to-do list além do básico. O usuário pode adicionar tarefas, marcá-las como concluídas, editá-las, excluí-las e ainda pesquisar ou filtrar pelo status — tudo sem precisar recarregar a página.

## ✨ Funcionalidades

- Adição de novas tarefas via formulário
- Edição de tarefas existentes com cancelamento da edição
- Marcação de tarefas como concluídas
- Exclusão de tarefas individualmente
- Pesquisa de tarefas em tempo real
- Filtro por status: **Todos**, **Feitos** e **A fazer**
- Ícones via Font Awesome

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome 6

## 🚀 Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Marcelo2707/to-do-list.git
   ```
2. Abra o arquivo `index.htm` no navegador.

Não é necessário instalar dependências ou rodar um servidor.

## 🖥️ Como funciona

- Digite a tarefa no campo de entrada e clique no botão **+** para adicioná-la.
- Clique no ícone de edição para alterar o texto de uma tarefa.
- Clique no ícone de check para marcar/desmarcar uma tarefa como concluída.
- Clique no ícone de lixeira para remover uma tarefa.
- Use o campo de **pesquisa** para filtrar tarefas pelo nome.
- Use o **select** de filtro para exibir apenas as tarefas feitas ou a fazer.
